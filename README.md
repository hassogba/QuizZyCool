QUIZZYCOOL est une petite application qui permet de créer des Tests à trous.

J'ai utilisé un modèle MVC pour écrire le code. Vous le trouverez sur le lien suivant:https://github.com/daveh/php-mvc.

Il suffit de créer un compte sur l'application de vous y connecter (http://quizzycool.epizy.com/public/).

Ensuite vous pouvez créer votre test, exporter votre test ou de passer un test enregistré.

Amusez-vous!
